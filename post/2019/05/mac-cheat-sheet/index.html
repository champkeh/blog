微调音量

单按 delete 删除光标之前的一个字符（默认）
组合 `fn + delete`，删除光标之后的一个字符
组合`option + delete`，删除光标之前的一个单词(英文有效)
组合`fn + option + delete`，删除光标之后的一个单词
组合`command + delete`，删除文件
